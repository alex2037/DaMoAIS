# Учебники

Базовые и дополнительные учебные материалы по проектированию, развёртыванию и сопровождению систем ИИ. Книги на английском доступны платно через издателей (O'Reilly, Питер) и в подписных электронных библиотеках; часть авторов сопровождает книги открытыми материалами (сайт, репозиторий). Официальная документация инструментов вынесена в раздел [«Прочее»](../other/README.md).

## Требования к добавлению

Для каждого учебника укажите автора, издателя и год, проверяемую устойчивую ссылку (страница издателя, сайт автора или DOI), связь с модулем/результатом обучения и условия доступа (платный/открытый). Не указывайте прямые ссылки на пиратские копии.
