# Банки задач

Практические, расчётные и кейсовые задания для лабораторных работ, СРС и сквозного проекта. Кейсы отражают прикладные сценарии эксплуатации ИИ-систем из компетентностно-ролевой модели (КРМ 3.0, см. [`data/`](../../data/README.md)); практические задания привязаны к модулям и доводятся до итогового проекта «shared ИИ-сервис на 30 пользователей». Материалы внутренние, распространяются вместе с репозиторием.
