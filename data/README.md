# Данные

Папка зеркалит слой данных интеллектуальной информационной системы (ИИС), которую студенты создают в дисциплине: здесь хранятся выгрузки компетентностной модели и вспомогательные наборы, на которые опираются оценочные средства. Учебные датасеты для лабораторных вынесены в [`resources/datasets/`](../resources/datasets/README.md).

## Состав

| Файл | Назначение |
|---|---|
| [`krm-v3.0.xlsx`](krm-v3.0.xlsx) | Выгрузка компетентностно-ролевой модели ИТМО (КРМ), версия 3.0: роли, компетенции (в т.ч. LC-5, BD-5), индикаторы и уровни (базовый/средний/продвинутый), кейсы. Источник истины для [модели измерения](../README.md#2-модель-измерения) и разметки индикаторов в РПД и КИМ. |

## Правила добавления данных

- Крупные бинарные файлы (> 10 МБ) в историю не коммитить — использовать Git LFS или ссылку на внешнее хранилище (см. [CONTRIBUTING.md](../CONTRIBUTING.md)).
- Для каждого набора указывать источник, версию и дату актуализации.
- Персональные данные и сведения ограниченного доступа не размещать.
