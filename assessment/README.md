# Фонд оценочных средств (ФОС)

> **Статус:** 📝 в работе — этап 2 проектной работы (старт 16.07.2026, после защиты РПД).
> Структура будет приведена к шаблону репозитория ФОС, который выдают организаторы ДПО.

ФОС разрабатывается по [рабочей программе дисциплины](../syllabus/README.md) и оформляется в виде Git-репозитория. Основа ФОС — модель измерения уровня освоения компетенций: через неё строится навигация по всем контрольно-измерительным материалам (КИМ), ресурсам и методическим указаниям.

## Модель измерения уровня освоения компетенций

Уровни — по КРМ (версия 3.0): **Б** (базовый), **С** (средний), **П** (продвинутый).
